# Thai Smartcard Reader with Node.js

## Installation

### Windows

- Install [Python 2.7.x](https://www.python.org/downloads/)
- Set enviroments

  ```text
  PATH=%PATH%;C:\Python27\Scripts\;C:\Python27\
  ```

- Run `$ npm install`

### Ubuntu & Pi

- Install [Python 2.7.x](https://www.python.org/downloads/)
- Run `$ sudo apt-get install libpcsclite1 libpcsclite-dev pcscd`
- Run `$ npm install`
